Letter an word Spacing

by TPS Sena

El kerning es el termino que utiliza los tipograficos para el espacio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: letter-spacing. Es particularmente útil para aumentar el kerning cuando su encabezado o frase está en mayúscula. Si el texto está en un caso o normal.

El intervalo predeterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra (a manudo alrededor de 0,25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
