Por favor, vea el origen de esta pagina para examinar los metadato

El elemento <meta;> reside dentro del elemento <meta;> y contiene informacion sobre esa pagina web. No es visible para los usaurios, pero cumple una serie de propositos, como decir a los motores de busquedad acerca de sus pagina, quien lo creo, y si es o no es sencible al tiempo. (Si la pagina es sensible al tiempo. se puede establecer para expiarar.) El elemento <meta;> es un elemento vacio por lo que no tiene etiqueta de cierre. Utiliza atributos para llevar la informacion. Los atributos mas comunes son el nombre y los atributos de contenido, que tieden a usarase juntos. Estos atributos especifican las propiedades de toda la pagian. El valor del atributo name es la propuedad que esta estableciendo y el valor del atributo content es el valor que deaeas dar a este propiedad.

description: contiene una description de la pagina. Esta description es comunmente utilizada por los motores de busquedad para entender de que se trata la pagina y debe ser un maximo de 155 carateres. A veces tambien se muesta en los resultados del motor de busquedad.

robots: esto indica si los motores de busquedad deben agregar esta pagina a sus resultados de busquedad o no. Se puede usar un valor de noindex so no se debe agregar a esta pagina. Un valor de nofollow se puede utilizar si los motores de busquedad deben agregar esta pagina en sus resultados, pero no las pahinas a las que enlaza.

author:define el autor de la pagina Web.

pragma: esto impide que el explorador almacenar en cache la pagina (Es decir, alamacenamiento localmente para ahorrar tiempo descargando visitas subsiguientes.)

expires: espira pr que los exploradores suelen almacerar en cache el contenido de una pagina, la opcion expires se puede usar para indicar cuando debe expirar la pagina (y ya no sea almacenada en cache.) Tenga en cuenta que la fecha debe especificarse en el formato que se muestra.
