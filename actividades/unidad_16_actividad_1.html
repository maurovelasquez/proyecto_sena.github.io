Conceptos clave en el posicionamiento de Elementos

Construnccion de bloques

CSS trata cada elemento HTML como si estuviera en su propio cuadro. los cuadros block-level comienzan en una nueva linea y actuan como los principales los bloques de Construnccion de cualquier diseño. mientras que los inline-box fluyen entre el texto circundante. puede controlar la cantidad de espacio que ocupa cada caja estableciendo el anco de los cuadros ( y a veces la altura tambien). para seprarar los cuadros, se puede usar border margenes, relleno y colores de fondo.

Normal flow

Cada elemento de block-level se encuentra en la parte superior de la siguiente. puesto que esta es la forma predeterminada en que los exploradores tratan los elementos HTML, no se necesita una propiedad Css para indicar que los elementos deben aparecer en el flujo normal.
