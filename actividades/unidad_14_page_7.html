La propiedad border-width se utiliza para contrilar el ancho de un borde.

Se utiliza los siguientes valores: thin, medium, thick.

Se puede especificar diferentes anchos para los cuatro valores de borde en una propiedad.


La propiedad display permite convertir un elemento insertado en un elemento de nivelde bloque o viceversa, y tambien se puede utilizar para ocultar un elemento de la pagina.
