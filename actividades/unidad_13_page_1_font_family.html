Font Family

by TPS Sena

La propueda font-family permite especifircar el tipo de letra que se debe utilizar para cualquier texto dento de la pagian web al que se aplica una regla CSS

Descripción

El valor de esta propieda es el nombre del tipo de letra que desea usar, La persona que visita el sitio web necesita que el tipo de letra que especificado se enuentre instalado en su equipo para que se muestre. Se puede especificar una lisra de fuentes separadas por comas de moda que, si el usuario no tiene instalada la primera opción de tipo de letra, el navegador pueda intentar utilizar una fuente alternativa de la lista.

Si un nombre de fuente se compone de más de una palabra, se debe poner entre comillas dobles. Los diseñadores sugieren que las paginas suelen lucir mejor si no utiliza más de tres tipografias en una página.
