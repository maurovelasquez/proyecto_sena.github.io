Floating elements-elementos flotantes

"la vida es como andar en bicicleta. para mantener el equilibrio debes seguir avanzando " Albert einstein

La propiedad float le permite tomar un elemento en el flujo normal y colocarlo tan lejos a la izquierda o a la derecha del elemento contenedor como sea posible, Cualquier otra cosa que se sienta dentro del elemento contenedor fluira alrededor del elemento que esta flotando

Cuando se utliliza la propiedad float, tambien se debe utilizar la propiedad width para indicar el ancho que debe tener el elemento flotante. si no se hace, los resultados pueden ser incoherentes, pero es problable que el cuadro tome el ancho completo del elemento contenedor (al igual que en el flujo normal)
