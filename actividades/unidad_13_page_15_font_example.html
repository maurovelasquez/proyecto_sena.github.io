Text Properties

by TPS Sena

Este ejemplo combina muchas de las tecnicas mostradas en los ejercicios

Hay propiedades para controlar el tipo de fuente, el tamaño, la ubicacion, el estilo y el espaciado.

Carateristicas

Puede controlar el espacio entre las líneas de texto, las letras indivuduales y las palabras. El texto tambien se puede alinear a la izquierda, derecha, centrada o justificada. Tambien puede tener sangria.
