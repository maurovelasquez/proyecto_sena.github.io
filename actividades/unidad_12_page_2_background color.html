Background Color - Color de fondo

Si no es especifica un color de fondo, el fondo es trasparante.

CSS trata cada elemento HTML como cuadro y la propiedad Background-color establece el color del fondo para el cuadro. Se puede especificar un color de fondo de las mismas tres formar en que puede especificar colores de primer plano: Valores RGB, Codigo hexadeximal y Nombre de colores. Tambien hemos utilizado la propieda de padding para separar el texto de los bordes de los cuadrados. Esto hace que sea mas facil de leer.
