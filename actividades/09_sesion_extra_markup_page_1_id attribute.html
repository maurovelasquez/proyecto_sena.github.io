Cada elemento HTML puede llevar el atributo id, se utiliza para identificar de forma unica ese elemento de otros elementos de la pagina. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carater). Es importante que no hay dos elementos en la misma pagina tiene el mismo valor para sus atributos ID (de lo contrario, el valor ya no es unico)

Dar a un elemento una identidad unica le permite darle un estilo diferente que cualquier otras instancia del mismo elemento en la pagina.

Por ejemplo, es posible que desean asignar un parrafo dentro de la pagina (quizas un parrafo que contiene una cotizacion de extraccion) un estilo difernte que todos los demas parrafos. en el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayuscula usando CSS.
