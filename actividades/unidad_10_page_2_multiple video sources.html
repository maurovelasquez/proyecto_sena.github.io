La etiquera video:Para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro del elemento <source> (Esto debe reemplazar el atributo src en la etiquera <source> de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el video esta disponible en diferentes formatos. (Debido a un error en el ipad, debe proporcionar el video en MP4 como el primer formato. De los cotrario puede que no incie.)

src:este atributo especifica la ruta de acceso del video.

type:se debe utilizar este atributo para decirle al navegador que formato es el video, de lo contrato, decargara parte del video para si se puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs:el codec que se uso para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, asi como comllas dobles en el atributo tipo, cuando suministra
