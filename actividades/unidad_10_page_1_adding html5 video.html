En HTML5 no es necesario proporconar valores para todos los atributos como los controles, reproduccion automatica y atributos de bucle uilizados con el elemento <video> Estos atributos son como innterructores de encendido/apagado. Si el atributo esta presente, activda esta opcion. Si se omite el atributo, la opción esta desactvidad. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrata lo que esta enntre <video> de apertura y las etuiquetas de <video> de cierre.

preload:este actributo le indica al nnavegador que hacer cuando se carga la pagina, puede tener uno de tres valores:
none:el navegador no debe cargar el video hasta que el usuario presione Play.
autoel navegador debe descargar el video cuando se carga la pagina.
metadata el navegador debe simplemente recopilar informacion como el tamaño, el primer fotograma, la lista de pista y la duracion.
